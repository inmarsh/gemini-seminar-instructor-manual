# Gemini ShopSupp プロンプト: イントロダクション

## 概要

Gemini ShopSuppは、キャリアショップ店員向けに特化した高度なAIプロンプトシステムです。このマニュアルでは、システムの基本的な使用方法から高度な機能まで詳細に説明します。

## 主な特徴

- 店員向けにカスタマイズされたAIアシスタント
- コンテキストに応じた柔軟な対応
- 高度な情報セキュリティ対策
- ユーザーエクスペリエンスの向上

## 対象読者

- キャリアショップ店員
- 店舗管理者
- ITサポートスタッフ
